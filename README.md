Portfolio et site vitrine de **Florian Philibert** — solutions logicielles sur mesure.

**🔗 Site live** : [fph-solutions.com](https://fph-solutions.com)

## Stack

| Technologie | Version |
|-------------|---------|
| Next.js | 16 |
| React | 19 |
| TypeScript | 5 |
| Tailwind CSS | 4 |
| Framer Motion | Dernière |
| Hébergement | Vercel |
| Domaine | Cloudflare |

## Projets présentés

- [Trackrma](https://trackrma.vercel.app) — Application SAV intelligente
- Gestion EPI — Gestion d'équipements professionnels
- SOBE Plomberie — Site vitrine pro
- Freshr — Application innovante
- ZenFlow — Productivité & bien-être
- Wedding Planner V2 — Planificateur de mariage

## Scripts disponibles

```bash
npm run dev     # Lancement en développement
npm run build   # Build production
npm run start   # Démarrage production
npm run lint    # ESLint
```

## Maintenance

- Les dépendances sont mises à jour automatiquement via Dependabot
- Le déploiement est automatique sur push vers `main` (Vercel)
- Monitoring via Vercel Analytics
